Technology Stack for Full Stack Web Developer (3+)
----------------------------------------------------

UI Technologies
----------------
HTML5 -> for structuring of the webpage
CSS3  -> for presentation of the webpage
JavaScript(ES6 to ES10) -> for producing Dynamic content using DOM Manipulation
JQuery -> Library of JavaScript for dynamic data like JS
AJAX -> for Asyncronous calls with server
Bootstrap4 -> for Responsiveness and Mobile friendly applications
Sass -> CSS Pre-processor
Less -> CSS Pre-processor
TypeScript3+ -> a superset of JavaScript with fully Object oriented
React JS -> for View layer in MVC architecture and one way data binding.
Redux -> Predictable State container for handling state data in React App.
Next JS -> for View layer in MVC architecture and one way data binding. An Extension to React JS.

Graphs & Charts
---------------
D3.js
C3.js

Optional Technologies
----------------------
React Native
Electron JS

Web Data Formats

JSON -> JavaScript Object Notation (latest mostly used with js)

Serverside Technologies
------------------------
Node JS -> for serverside JavaScript for serverside processing
Express JS -> Library of Node JS for serverside processing
